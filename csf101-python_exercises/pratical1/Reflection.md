Practical 1 Reflection
Instructions
For this practical, we were required to:

Explore and implement Python basics, including numbers, strings, booleans, and type conversion.
Practice using operators (arithmetic, assignment, comparison, logical).
Apply control structures such as conditionals, loops, and control statements (break, continue).
Write and use functions, understand parameters, return values, recursion, and variable scope.
Work with data structures like lists and dictionaries.
Perform file operations (reading/writing text and binary files, file management).
Develop algorithmic thinking by creating flowcharts and pseudocode before coding.
Each topic was divided into sub-tasks, and we were expected to write Python scripts for each, test them, and reflect on the learning process.

Solutions
Basics
numbers.py: Demonstrated integer and float operations, type casting, and basic math functions.
strings.py: Showed string creation, concatenation, slicing, and formatting.
booleans.py: Used boolean values in expressions and logical operations.
type_conversion.py: Converted between types (int, float, str, bool) and handled conversion errors.
Operators
arithmetic.py: Used addition, subtraction, multiplication, division, modulus, exponentiation, and floor division.
assignment.py: Applied assignment operators (=, +=, -=, etc.).
comparison.py: Compared values using ==, !=, <, >, <=, >=.
logical.py: Combined conditions using and, or, not.
Control Structures
conditionals.py: Wrote if, elif, else statements for decision making.
loops.py: Used for and while loops to iterate over sequences and ranges.
break_continue.py: Demonstrated breaking out of loops and skipping iterations.
Functions and Scope
basic_functions.py: Defined and called simple functions.
parameters_and_returns.py: Used parameters and return values in functions.
recursion.py: Implemented recursive solutions (e.g., factorial, Fibonacci).
scope.py: Explored local and global variable scope.
Data Structures
lists.py: Created, modified, and iterated through lists; performed slicing and sorting.
dictionaries.py: Created dictionaries, accessed/modified key-value pairs, iterated through items.
File Operations
text_files.py: Read from and wrote to text files, handled file modes and exceptions.
binary_files.py: Worked with binary files for reading and writing data.
file_management.py: Checked file existence, deleted files, and managed file paths.
FlowChart & Pseudocode
Created flowcharts and pseudocode for selected problems before implementing them in Python, which helped clarify logic and structure.
